Typeasy

Typesay is web software you can use to create a blog which is simple and clean.

View project onGitHub

Typeasy是一个使用Struts2框架编写的轻量级博客程序.

详细介绍请参阅：http://llss.me/typeasy

使用方法：

1、点击右侧的

2、将document文件夹下的应用

3、导入document文件夹下的typeasy.sql数据库

4、修改DBConfig.properties中的数据库连接

5、开始使用
